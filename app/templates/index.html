交通监控系统 API

系统简介

本系统使用 EfficientNet 深度学习模型进行交通图像分析。 可以识别多种交通状况，包括：

  • 交通事故 (accident)
  • 交通拥堵 (dense)
  • 交通火灾 (fire)
  • 交通顺畅 (smooth)

API 服务基于 FastAPI 构建，提供高性能、易用的 REST API。

API 示例

      
使用 curl 发送预测请求：

      
curl -X POST \
  http://localhost:8000/api/v1/predict \
  -H "Content-Type: multipart/form-data" \
  -F "file=@/path/to/image.jpg" \
  -F "model_name=efficientnet"
